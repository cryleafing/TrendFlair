TrendFlair is a demo web application designed to simulate an engaging, modern online thrift-shopping experience. This project focuses on responsive design, high-quality visual presentation, and interactive UI elements.

## Features

Responsive Design: Fully optimised for mobile, tablet, and desktop devices using CSS Grid.

Dynamic Product Catalog: Features a curated collection of clothing items.

Interactive UI: Includes custom CSS styling, such as the "Start Thrifting" call-to-action button featuring a subtle glow effect.

## Tech Stack

HTML5

CSS3 (Flexbox/Grid, Animations)

JavaScript (ES6+)

## Installation

1. Clone the repository:

Bash
git clone https://github.com/yourusername/trendflair.git

2.  Navigate to the project directory:

bash
cd trendflair

3. Open index.html in your preferred web browser to view the demo.
